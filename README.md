# E-Shop Tech

Single Page Application (React) pour une boutique de produits informatiques.

## Fonctionnalites

- Navigation multi-pages avec React Router (Accueil, Detail, Panier, Commande, 404).
- Gestion du panier via Redux Toolkit (ajout, suppression, quantite, total).
- Consommation d'une API publique avec etats Loading et Error.
- Formulaire de commande avec validation.

## Demarrer

- Installer les dependances: `npm install`
- Lancer en developpement: `npm run dev`
- Build de production: `npm run build`
- Previsualiser le build: `npm run preview`
